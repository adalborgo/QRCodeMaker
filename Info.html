QRCode Maker

L'applicazione QRCode Maker dà la possibilità di generare immagini di QR Code a partire da diversi tipi di input: la scelta viene fatta con la selezione di Modo: 1. Testo; 2. File di testo; 3. File di archivio.

Modo "Testo"

La stringa da trasformare in QR Code viene inserita da tastiera.

Testo: scrivere il testo da codificare: si possono anche copiare i dati nell’area di testo tramite la combinazione di tasti Ctrl-C e Ctrl-V.

File immagine: indicare il nome del file con l’immagine.

 

Modo "File di testo"

La stringa da trasformare in QR Code viene caricata da un file di testo.

File dati: indicare il file contenente il testo da trasformare in QR Code.

File immagine: indicare il nome del file che deve contenere l’immagine QR Code.

 

Modo "File di archivio"

Questa modalità è particolarmente utile per generare più QR Code a partire da un unico file. Ogni riga del file sorgente dovrà contenere sia il testo da codificare sia il nome del file immagine separati dal carattere “|”; il nome del file immagine può essere omesso, in questo caso i file immagine avranno un numero progressivo.

File dati: indicare il file sorgente.

Intestazione: è facoltativa ma può essere utile da aggiungere come costante a ciascun testo.

Cartella: dove devono essere archiviati i file immagine.
